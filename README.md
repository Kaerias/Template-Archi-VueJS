# Viticia WebSite

## Project setup

```
npm ci
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Run your tests

```
npm run test
```

### Run the docker

```
docker-compose up --build
```

### Lints and fixes files

```
npm run lint
```
